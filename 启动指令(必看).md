## 启动命令

1. pkg update && pkg upgrade

2. pkg install openssl-tool

3. pkg install python

4. pkg install git

5. pkg install python -y

6. pip install pycryptodome

7. git clone https://github.com/xiaodai56/ToT.git

8. cd ToT

9. find ~/ToT -name .giteepk -delete

10. chmod +x XD/XDTOOT

11. chmod +x XD/AddExecutePermission

12. chmod +x XD/SetupEnv

13. ./XD/XDTOOT
