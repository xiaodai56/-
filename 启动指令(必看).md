## 启动命令

1. pkg update && pkg upgrade

2. pkg install python

3. pkg install git

4. pkg install python -y

5. pip install pycryptodome

6. git clone https://github.com/xiaodai56/ToT.git

7. cd ToT

8. find ~/ToT -name .giteepk -delete

9. chmod +x XD/XDTOOT

10. chmod +x XD/AddExecutePermission

11. chmod +x XD/SetupEnv

12. ./XD/XDTOOT
