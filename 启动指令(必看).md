## 启动命令

1. pkg install python

2. pkg install git

3. git clone https://github.com/xiaodai56/ToT.git

4. cd ToT

5. find . -name ".gitkeep" -type f -delete

6. chmod +x tools/Menu

7. chmod +x tools/AddExecutePermission

8. chmod +x tools/SetupEnv

9. ./tools/Menu
